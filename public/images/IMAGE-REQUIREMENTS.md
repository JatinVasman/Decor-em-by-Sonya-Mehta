## Image Guidelines

### Color Palette
All images should have warm tones matching:
- Warm whites (#FAF9F6, #E4DED7)
- Gold accents (#C9A76F)
- Deep contrasts (#1E2A36, #2B2B2B)

### Technical Specs
- Format: JPG or WebP
- Quality: 85-90%
- Dimensions: Minimum as specified above
- File size: Optimize for web (< 500KB per image)
- Include proper alt text for SEO

### Style
- Professional photography
- High-end luxury aesthetic
- Warm, inviting lighting
- Clean, minimal compositions
- Focus on details and craftsmanship
